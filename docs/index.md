# Arilekh Report

Welcome to **Arilekh Report**.

Arilekh Report is a cross-platform reporting engine for .NET applications.

## Features

- Report Designer
- Report Viewer
- Charts
- Expressions
- Export to PDF
- Export to Excel
- Export to HTML

## Supported Platforms

- .NET 8+
- ASP.NET Core
- Windows
- Linux
- Docker
